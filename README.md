## AdChina

> 技术架构：使用react+react-router+redux+less+ES6+webpack实现;


> 在线演示地址：[https://www.adchina.club](https://www.adchina.club)

### 功能说明

广告推送服务，让企业与商户互帮互赢




### 本地演示方法

```text
# 安装依赖
npm install
# 本地运行
npm run start
# 导出文件到build文件夹
npm run build
# 运行地址
localhost:3000
```
